# romdevsetup

  - Initial rom development environment setup shell script.

### Installation

> Only works on ubuntu distros and run these commands as root

Installation.

```sh
$ apt-get update
$ apt-get install git
$ git clone https://github.com/jobinrjohnson/romdevsetup.git
$ cd romdevsetup
$ chmod +x romsetup.sh
$ ./romsetup.sh
```
